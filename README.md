# Mi Página Web Personal

Este proyecto es una página web personal desarrollada con HTML y CSS básico.  
Incluye información sobre mí, mis intereses, formación y un formulario de contacto.

## Autor
Gregorio Emmanuel Garzón Morla

## Contenido
- Encabezado con menú de navegación
- Sección "Sobre mí"
- Sección de intereses con lista
- Formación y conocimientos
- Formulario de contacto

## Tecnologías
- HTML
- CSS

## Instrucciones
1. Abrir `index.html` en cualquier navegador.
2. Revisar la navegación interna usando el menú.
3. Verificar que las imágenes se muestren correctamente desde la carpeta `img/`.
